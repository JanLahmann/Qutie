# Pictures

Drop photos of printed Quties (`.jpg`, `.png`, `.webp`, …) into this folder.

Every picture here automatically appears in the gallery on [qutie.org](https://qutie.org),
and the first one (alphabetically) is used as the hero photo. The filename becomes the
caption: `Qutie-on-a-desk.jpg` shows up as **“Qutie on a desk”**.
