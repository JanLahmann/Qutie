# STL files

Drop the 3D-printable Qutie models (`.stl`) into this folder.

Every file here is automatically published on [qutie.org](https://qutie.org) with an
interactive 3D preview and a download button. The filename becomes the display name:
`Qutie-System-Two.stl` shows up as **“Qutie System Two”**.
